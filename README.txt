Installation :
1. Décompresse le ZIP à la racine du projet.
2. Accepte le remplacement de src/pages/More.jsx.
3. Vérifie que src/components/more existe.
4. Redémarre Vite avec :
   Ctrl + C
   npm run dev
5. Recharge avec Ctrl + F5.

Aucune modification CSS nécessaire.
